🚀 COMO USAR:

1. BANCO DE DADOS:
   - Abra: docs/PROMPT-BANCO-DADOS.txt
   - Copie o prompt
   - Cole em um AI (Claude, ChatGPT, etc)
   - Copie o SQL gerado
   - Abra MySQL Workbench
   - File > New Query Tab
   - Cole e execute o SQL

2. BACKEND:
   cd backend
   npm install
   npm start
   
   Verá: 🚀 Servidor rodando em http://localhost:5000

3. FRONTEND (novo terminal):
   cd frontend
   npm install
   npm run dev
   
   Verá: ➜ Local: http://localhost:5173

4. NAVEGADOR:
   Abra http://localhost:5173
   Crie uma conta e teste!
